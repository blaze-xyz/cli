# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email us at **security@blaze.money** with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

We will acknowledge your report within 48 hours and work with you to understand and address the issue promptly.

## Supported Versions

Security updates are applied to the latest release. We recommend always running the most recent version.
