# Examples

These examples demonstrate common usage patterns for the Blaze SDK.

## Running Examples

Set your API key and run any example with `tsx`:

```bash
BLAZE_API_KEY=sk_test_... npx tsx examples/sdk-quickstart.ts
```

## Available Examples

- **sdk-quickstart.ts** -- Check balance and list customers.
- **send-money.ts** -- Find or create a customer and initiate a transfer.
- **create-payment-link.ts** -- Create a payment link for invoicing.
- **list-with-pagination.ts** -- Iterate through paginated results.
